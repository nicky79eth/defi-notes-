# MakerDAO

MakerDAO is a decentralized protocol that issues the DAI stablecoin.

DAI is a decentralized stablecoin designed to maintain a value close to 1 USD.

## How It Works

Users lock collateral (such as ETH) into smart contracts called vaults.

In return, they can generate DAI.

## Key Components

- DAI stablecoin
- collateralized vaults
- governance by MKR token holders
