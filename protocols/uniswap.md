# Uniswap

Uniswap is a decentralized exchange (DEX) protocol that allows users to swap tokens directly from their wallets.

It uses an Automated Market Maker (AMM) model instead of a traditional order book.

## Key Features

- permissionless trading
- liquidity pools
- automated pricing

## How It Works

Liquidity providers deposit token pairs into pools.

Traders swap tokens against these pools, and prices are determined by the constant product formula:

x * y = k

## Benefits

- decentralized trading
- no intermediaries
- open access
