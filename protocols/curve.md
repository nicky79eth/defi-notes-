# Curve Finance

Curve is a decentralized exchange optimized for stablecoin trading.

It uses a specialized AMM designed to reduce slippage when swapping assets with similar prices.

## Key Features

- low slippage for stablecoins
- efficient liquidity pools
- yield opportunities for liquidity providers

## Common Pairs

- USDT / USDC
- DAI / USDC
- ETH / stETH
