# Flash Loan Attack

Flash loans allow users to borrow large amounts of assets without collateral as long as the loan is repaid in the same transaction.

While useful, flash loans can be exploited in attacks.

## How Flash Loans Work

1. Borrow assets from a liquidity pool
2. Use funds in arbitrage or other operations
3. Repay the loan within the same transaction

## Flash Loan Attacks

Attackers can manipulate price oracles or exploit vulnerabilities using large borrowed capital.

## Risks

- price manipulation
- oracle attacks
- liquidity pool exploitation
